== Installation ==

To install the plugin, follow these steps:

1. Upload the plugin files to the `/wp-content/plugins/nuway` directory, or install the plugin through the WordPress plugins screen directly.
2. Activate the plugin through the 'Plugins' screen in WordPress.
3. Click on the "Nuway" link on the left-side menu to configure the plugin.
4. Enter your CompanyID from the page in the Nuway dashboard panel (https://app.nuway.co/#/business/setting/chatbot).
5. Save the settings, then you can see the Nuway widget on your webpages.
